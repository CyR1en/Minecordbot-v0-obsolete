<h3 align= "center">Know more about Minecordbot at www.minecordbot.com</h3>
<hr />
<p align= "center">&nbsp;</p>
<p><strong>Feature:</strong></p>
<ul>
<li><b>Bind Minecraft and Discord chat.</b></li>
<li><b>Easy to set up.</b></li>
<li><b>Bot account manager.</b></li>
<li><b>Light weight.</b></li>
<li><b>Do Minecraft commands from discord.</b></li>
<li><b>Configurable broadcasts for Minecraft events.</b></li>
<li><b>Supports Multiple bound channels&nbsp;</b></li>
<li><b>Shows basic info about the server.</b></li>
</ul>
<p><b>(to learn more go check out the commands <a title="c" href="https://minecordbot.com/commands/">here</a>)</b></p>
